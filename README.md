# Portfolio Project Structure

## Directory Layout

```
/
├── index.html              # Main HTML file (modular)
├── style.css               # External CSS stylesheet
├── script.js               # External JavaScript
├── assets/
│   ├── images/             # Image files directory
│   │   └── (your images here)
│   └── resume.pdf          # Resume PDF file
└── README.md               # This file
```

## How to Use

1. **index.html** - Main entry point that links to external CSS and JS
2. **style.css** - All CSS styling (separated from HTML)
3. **script.js** - All JavaScript functionality (separated from HTML)
4. **assets/images/** - Store all image files here
5. **assets/resume.pdf** - Add your resume PDF file here

## Benefits of Modular Structure

✅ **Better Maintainability** - CSS and JS are in separate files  
✅ **Improved Performance** - Browser caching for CSS/JS files  
✅ **Cleaner Code** - Easier to read and debug  
✅ **Scalability** - Easy to add more images, PDFs, or other assets  
✅ **Professional Practice** - Follows web development best practices

## Quick Links

- Update the resume link: `assets/resume.pdf`
- Add images to: `assets/images/`
- Modify styles in: `style.css`
- Update functionality in: `script.js`
